# Image Clustering with VGG16 and UMAP 

## Обзор

Проект посвящён **извлечению признаков изображений с помощью сверточной нейронной сети (VGG16)** и их **визуализации в низкоразмерном пространстве** с использованием алгоритма UMAP.

Основная цель — показать, как нейросеть автоматически формирует **осмысленные представления изображений**, и как эти представления группируются по классам.

---

## Задачи проекта

В рамках работы были выполнены:

1. Визуализация активаций сверточных слоев VGG16  
2. Загрузка и обработка датасета изображений  
3. Извлечение признаков (эмбеддингов)  
4. Снижение размерности признакового пространства  
5. Визуализация кластеров изображений  

---

## Используемые методы

- **CNN (VGG16, pretrained on ImageNet)**  
- **Feature extraction (transfer learning)**  
- **UMAP (manifold learning)**  
- **Визуализация признаков**
